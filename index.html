// 01 Descripción del Curso

Ingeniería de Datos
de Alto Nivel

Un viaje estructurado por los fundamentos avanzados de las bases de datos modernas.

El curso de Base de Datos II profundiza en el diseño, implementación y optimización de sistemas de gestión de bases de datos relacionales y no relacionales. A lo largo de 4 unidades distribuidas en 16 semanas, explorarás desde consultas SQL avanzadas hasta arquitecturas distribuidas y big data.

Cada semana cuenta con recursos académicos descargables, prácticas en SQL y podcast temáticos que refuerzan el aprendizaje. El repositorio se actualiza constantemente con material exclusivo del curso.
